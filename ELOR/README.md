# EloR — International Rugby Rating System & Dashboard

Elo-based rating system for international rugby union. Covers matches from 1890 to present. Includes a Streamlit dashboard and CLI predictor.

Publication: [Northwestern MSIA Blog, January 2019](https://sites.northwestern.edu/msia/2019/01/25/introducing-a-new-rating-system-for-world-rugby-union-based-on-the-elo-rating-system-the-elor-elo-rugby/)

## Files

| File | Description |
|------|-------------|
| `dashboard.py` | Streamlit dashboard — 6 analytical tabs (see below) |
| `elo_over_time.py` | Builds `elo_history.csv` — ELO per team per match |
| `era_comparison.py` | Builds `era_rankings.csv` — average ELO by historical era |
| `upsets.py` | Builds `upsets.csv` — top 50 upsets ranked by upset probability |
| `predictor.py` | CLI match predictor — win probabilities for any two teams |
| `fetch_results.py` | Fetches latest match results from the World Rugby API |
| `update_data.py` | Runs the full data refresh pipeline |
| `functions.py` | Core ELO calculation functions |
| `main.ipynb` | Main ELO processing notebook |
| `expectedwins.ipynb` | Expected vs actual wins analysis |

## Datasets/

| File | Description |
|------|-------------|
| `ELO.csv` | Full match history with ELO ratings (primary dataset) |
| `get_rank.csv` | Current team rankings |
| `performance.csv` | Expected vs actual wins by team-year |
| `fixtures.csv` | Match fixture data |
| `results.csv` | Match results |
| `winper.csv` | Win percentages by team |

Generated outputs (created by running the scripts above):

| File | Generated by |
|------|-------------|
| `elo_history.csv` | `elo_over_time.py` |

## Setup

```bash
pip install -r requirements.txt
```

## Execution Order

Run these steps in sequence. Steps 1–3 only need to be re-run when you want to refresh the data.

> All scripts must be run from the `ELOR/` directory.

**Step 1 — Fetch latest match results:**
```bash
python fetch_results.py
```
Appends new matches from the World Rugby API to `Datasets/fixtures.csv`.
Use `python fetch_results.py --replace` for a full rebuild from scratch.

**Step 2 — Recalculate ELO ratings:**
Open `main.ipynb` and run all cells.
Outputs: `Datasets/ELO.csv`, `Datasets/get_rank.csv`, `Datasets/performance.csv`, `Datasets/winper.csv`

> `update_data.py` runs Step 1 automatically and then prints a reminder to run main.ipynb.

**Step 3 — Build dashboard data files:**
```bash
python elo_over_time.py   # Outputs: elo_history.csv
python era_comparison.py  # Outputs: era_rankings.csv
python upsets.py          # Outputs: upsets.csv
```

**Step 4 — Launch the dashboard:**
```bash
streamlit run dashboard.py
```

**Optional — Predict a match from the command line:**
```bash
python predictor.py --home "New Zealand" --away "South Africa"
python predictor.py --home "Ireland" --away "England" --neutral
```

## Dashboard Tabs

1. **Rankings** — current ELO ratings, top 30 bar chart, search/filter
2. **ELO Over Time** — historical rating trajectories by team and year range
3. **Era Dominance** — average ELO by era (Pre-WW1, Interwar, Post-WW2, Professional, Modern)
4. **Greatest Upsets** — top 50 upsets ranked by upset score (0.5 − winner probability)
5. **Match Predictor** — interactive win probability calculator with adjustable home advantage
6. **Performance** — over/under performers: actual wins vs ELO-expected wins
